# 无限细化方法论 — 全局元指令 (Meta Instructions)

## 核心原则 (Core Directives)

1. **绝对原真性 (Zero-Tampering)**
   你在执行任何文本处理、标签插入时，绝对**禁止**增、删、改底本文本中的任意一个汉字或标点符号。
   你只被允许在文中**插入**特殊的标记符号（如 `〖@名宦:xxx〗`）。一旦去除你所有插入的符号，剩下的文本必须与原始输入保证字节级一致！

2. **禁止幻觉补全 (No Hallucination)**
   - 对于不确定的年代（如只写“三年”），除非上下文绝对明确，否则**禁止**擅自折算为公元纪年，应暂存为 `null` 或原文表述，交由后续阶段处理。
   - 对于不确定的地理坐标，**绝对禁止**擅自给出经纬度。所有空间换算由专门的 Python 脚本对接 CHGIS 数据库处理。

3. **单点聚焦 (Single Skill Focus)**
   严格遵守当前的阶段任务指令。让你提取实体，你只打标签，绝不做 JSON 格式化；让你做 JSON 结构化，绝不再去检查是否有遗漏的实体。

4. **历史知识库辅助**
   如果你确信古文中某别名、字号对应当朝某特定历史人物（但原文中未点明本名），你可以通过符号内的规范名字表达出来：如原文写“东坡”，你标注为 `〖@名宦:苏轼〗`，但外部原文“东坡”二字不可变，最终组合为 `〖@名宦:苏轼〗东坡`。
