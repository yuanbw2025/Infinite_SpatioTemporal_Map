# 无限细化方法论 — 全局元指令 (Meta Instructions v3 · XML)

适用于：史传（《史记》《汉书》等纪传体）、方志（府志、县志）、笔记（《传习录》等）。

## 一、核心原则 (Core Directives)

### 1. 绝对原真性 (Zero-Tampering)
在实体标注阶段，你**只被允许插入** XML 开/闭合标签（形如 `<p>...</p>`、`<loc>...</loc>`），**禁止增、删、改底本中任何一个汉字或标点**。
一旦对输出执行 `re.sub(r'</?[a-zA-Z][^>]*>', '', output)`，剩下的文本必须与原始输入**字节级一致**。
若不一致，批次将被系统拒收。

> **重点**：标签是"括在原文外面"，不是"替代原文"。标签里留原字，canonical 写在属性里。
> 正确：`<p name="项羽" sub="名将">项籍</p>`
> 错误：`<p name="项羽" sub="名将">项羽</p>项籍`（这样会在 strip 后重复原文 → tamper 失败）

### 2. 禁止幻觉补全 (No Hallucination)
- **年代**：原文写"三年"、"其九月"等相对时间时，若无上下文明示具体帝王或年号，**禁止**擅自换算成公元纪年；JSON 阶段允许字段为 `null` 或保留原文 `"raw": "秦二世元年九月"`。
- **经纬度**：**绝对禁止**自行给坐标。所有空间换算由 Python 对接 CHGIS v6 shapefile 处理。
- **人物身份**：原文只有"籍"、"沛公"、"项王"等时，如能 100% 确认指代，可在 `name=""` 属性里写规范名；**不确定就空着属性或干脆不标**。
- **事件细节**：原文没提到的战役人数、伤亡、动机等，一律不得编造。

### 3. 单点聚焦 (Single Skill Focus)
每个 skill 文件只做一件事。
- `01_entity_annotation` 只插入 XML 标签，不结构化。
- `02_json_extraction` 只结构化，不再挖新实体。
- `03_reflection_check` 只挑错，不动结果。

### 4. 别名解析 (Alias Resolution)
允许使用标准名替换别名，但**只在标签的属性中写**，正文原字不能变。
例：原文"东坡" → `<p name="苏轼" sub="文臣">东坡</p>`
例：原文"项王" → `<p name="项羽" sub="名将">项王</p>`

### 5. 置信度声明 (Confidence)
JSON 阶段每个实体/事件必须含 `confidence ∈ {"high","medium","low"}`：
- high：原文直接点名，无歧义。
- medium：需上下文推断，但依据充足。
- low：推断链条较长。

## 二、输出格式硬约束
- 标注阶段：必须返回**纯文本**（含 XML 标签），不要任何解释、寒暄、Markdown 代码围栏、"以下是…"。
- JSON 阶段：必须以 ```` ```json ```` 开头、```` ``` ```` 结尾，中间是合法 JSON，**其他一律不要**。

## 三、XML 标签速查（详见 01）

| 类别 | 标签 | 示例 |
|---|---|---|
| 人物 | `<p name="规范名" sub="子类">...</p>` | `<p name="项羽" sub="名将">项籍</p>` |
| 行政地名 | `<loc>...</loc>` | `<loc>彭城</loc>` |
| 山川水域 | `<geo>...</geo>` | `<geo>乌江</geo>` |
| 官职 | `<off>...</off>` | `<off>上将军</off>` |
| 时间 | `<time>...</time>` | `<time>秦二世元年七月</time>` |
| 事件 | `<evt name="规范名">...</evt>` | `<evt name="巨鹿之战">破釜沉舟</evt>` |
| 文物 | `<art>...</art>` | `<art>玉玦</art>` |
| 遗址 | `<ruin>...</ruin>` | `<ruin>鸿门</ruin>` |

## 四、绝不做的事
1. 不要翻译、意译、加白话注释。
2. 不要改简繁字形（底本什么字形就什么字形）。
3. 不要合并/拆分段落。
4. 不要输出"以下是结果"之类的场面话、不要加 ```` ``` ```` 围栏。
5. 不要在标签内嵌套另一个标签（不要写 `<p>A<loc>B</loc>C</p>`）。
6. 不要在标签的"标签体内容"里重复写一次原文字符以外的内容。**tag 体必须是原文连续子串**。
