# 技能 01 — 史传/方志 实体标注 (Entity Annotation v3 · XML)

## 任务
读一段文言原文，**只插入** XML 标签（开标签 + 闭标签）把历史实体括起来。**不得改动任何一个汉字/标点/空白**。

## 硬约束（违反即拒收）

1. **Zero-Tampering**：
   对输出执行 `re.sub(r'</?[a-zA-Z][^>]*>', '', output)`，结果必须与原文**字节级一致**。
   → 含义：**标签体 `<tag>...</tag>` 中间的内容必须是原文的连续子串**；你不能写 `<p name="项羽">项羽</p>项籍`，要写 `<p name="项羽">项籍</p>`。

2. **不嵌套、不重叠**：标签体内部不得再出现任何 `<…>`。

3. **属性用双引号**：`name="项羽" sub="名将"`，不要单引号。

4. 拿不准的实体**宁可不标**。

5. **只返回标注后的正文**，前后不要解释、不要代码围栏、不要"以下是…"。

## 允许的标签

### A. 人物 `<p name="规范名" sub="子类">原字</p>`
子类 `sub` 固定取以下之一：
- `帝王`（刘邦、义帝、楚王、秦始皇、秦二世…自立或被立之王皆可）
- `名将`（项羽、章邯、韩信、王翦、英布、彭越）
- `谋士`（范增、张良、陈平）
- `宗室`（项伯、项庄、项梁等主角亲属）
- `文臣`（萧何、周苛）
- `女性`（虞姬、吕后）
- `寒士`（田父、亭长、无名氏）
- `人物`（兜底）

规范名建议：
- 原文"项羽/项王/籍/项籍" → `name="项羽"`
- 原文"沛公/汉王/刘邦" → `name="刘邦"`（项羽本纪中沛公尚未称帝，但通名用"刘邦"）
- 原文只有字号（东坡/子瞻）→ `name="苏轼"`
- 原文是小角色（"田父"、"乌江亭长"）→ 就把原称呼当 name。

### B. 行政地名 `<loc>原字</loc>`
行政区、城邑、关隘。
例：`<loc>下相</loc>` / `<loc>彭城</loc>` / `<loc>鸿门</loc>` / `<loc>函谷关</loc>` / `<loc>咸阳</loc>` / `<loc>江东</loc>` / `<loc>垓下</loc>`（垓下作地标按 loc）

### C. 山川水域 `<geo>原字</geo>`
明显是水/山。例：`<geo>乌江</geo>` / `<geo>淮水</geo>` / `<geo>浙江</geo>`

### D. 官职 `<off>原字</off>`
例：`<off>上将军</off>` / `<off>裨将</off>` / `<off>楚将</off>` / `<off>秦将</off>` / `<off>会稽郡守</off>` / `<off>霸王</off>`（自封号也算）

### E. 时间 `<time>原字</time>`
完整的时间短语都要标。
例：`<time>秦二世元年七月</time>` / `<time>汉之三年</time>` / `<time>汉五年</time>`
**不换算公元**。

### F. 事件 `<evt name="规范名">原字</evt>`
只标有传统命名的战役/政变，指示性短语即可，不要把整段叙事都包进去。
例：`<evt name="大泽乡起义">起大泽中</evt>` / `<evt name="巨鹿之战">破釜沉舟</evt>` / `<evt name="鸿门宴">鸿门</evt>`（谨慎，不要与 loc 冲突；若整句就是事件名对应短语才标）

### G. 其他
- `<art>原字</art>`：文物（玉玦、白璧、玄甲）
- `<ruin>原字</ruin>`：遗址（项王谷城、新安坑秦处）

## Few-Shot（项羽本纪）

### 示例 1（人物 + 地名 + 官职）
**输入：**
```
项籍者，下相人也，字羽。初起时，年二十四。其季父项梁，梁父即楚将项燕，为秦将王翦所戮者也。
```
**输出：**
```
<p name="项羽" sub="名将">项籍</p>者，<loc>下相</loc>人也，字羽。初起时，年二十四。其季父<p name="项梁" sub="宗室">项梁</p>，梁父即<off>楚将</off><p name="项燕" sub="名将">项燕</p>，为<off>秦将</off><p name="王翦" sub="名将">王翦</p>所戮者也。
```
自检：去掉所有 `<...>` 标签后 → "项籍者，下相人也，字羽。初起时，年二十四。其季父项梁，梁父即楚将项燕，为秦将王翦所戮者也。" ✅

### 示例 2（时间 + 事件 + 人物 + 地名）
**输入：**
```
秦二世元年七月，陈涉等起大泽中。
```
**输出：**
```
<time>秦二世元年七月</time>，<p name="陈涉" sub="人物">陈涉</p>等<evt name="大泽乡起义">起大泽中</evt>。
```
> 注意：`<evt>` 体是 `起大泽中`（原文 4 字连续子串），**不要写成** `<evt name="大泽乡起义">陈涉起大泽中</evt>`，因为"陈涉"已经被 `<p>` 占了，不能嵌套/重叠。

### 示例 3（鸿门宴，多人物别称混合）
**输入：**
```
沛公旦日从百余骑来见项王，至鸿门，谢曰：
```
**输出：**
```
<p name="刘邦" sub="帝王">沛公</p>旦日从百余骑来见<p name="项羽" sub="名将">项王</p>，至<loc>鸿门</loc>，谢曰：
```

### 示例 4（拿不准，不标）
**输入：**
```
诸将皆曰：“善。”
```
**输出：**
```
诸将皆曰：“善。”
```

### 示例 5（死亡+地点）
**输入：**
```
项王乃自刎而死于乌江之上。
```
**输出：**
```
<p name="项羽" sub="名将">项王</p>乃自刎而死于<geo>乌江</geo>之上。
```

## 自检清单（每段输出前默做一遍）
1. 所有 `<xxx>` 是否都有对应的 `</xxx>`？
2. `<p>`/`<loc>`/`<geo>`/`<off>`/`<time>`/`<evt>`/`<art>`/`<ruin>` 体内部是否**全是原文连续子串**，**没有**出现"规范名的复制"或"新增解释"？
3. 标签体内是否再出现 `<` 或 `>`？（不允许）
4. 心算 strip：把所有 `<[^>]+>` 删掉，剩的是不是原文一字不差？
