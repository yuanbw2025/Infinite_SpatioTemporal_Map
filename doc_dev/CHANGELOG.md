## 2026-04-21

### 确立数据管线双轨架构
- **决策**：数据提取管线支持两种运行模式，共享同一套提示词和后处理脚本
  - **模式 A（订阅/对话模式）**：脚本拆 PDF → 生成分批页面图片包 + 对应提示词 → 用户粘贴至 AI Studio / Claude 对话 → 复制模型输出 → 脚本解析 JSON
  - **模式 B（API 自动模式）**：脚本拆 PDF → 自动调用 LLM API 处理 → 脚本解析 JSON
- **理由**：用户不希望仅依赖 API 付费模式，需要能复用订阅额度；两种模式输出格式一致，数据可无缝合并
- **注意**：每批建议处理 10 页（对话模式受上下文限制），API 模式可整本处理

### 新增开发文档
- **新建** `AI_HANDOVER.md`：技术交接手册，精确到文件路径和 JSON 字段名，确保任何模型接手时技术方向一致
