# 无限时空图工程实施计划 (Phased Execution Plan)

根据我们在《核心研发主控规划》中敲定的路线，实际开发的落地执行将严格划分为以下四个批次。我们将像修金字塔一样，从底层承重墙一路搭到最顶层的展示厅。

## 实施阶段拆解

### 🧱 Phase 0: MVP 破冰与前端 3D 沙盘点亮
*当前状态：就绪，随时开工。*

**核心目标**：绝不搞后端依赖。防患于未然，优先测试纯前端极限渲染 3D 版图和底片的框架通不通。
**具体拆解动作**：
1. **搭建底座**：通过 Vite 初始化标准的 Vue 3 空骨架。
2. **引入引擎**：将核心的无闭源依赖地图框架（MapLibre GL JS）和特效渲染引擎（deck.gl）装配到 Vue 项目中。
3. **写死定点测试**：手工攥造 3 个含有 [经纬度, 朝代时间] 的模拟虚构历史坐标流数据（JSON Mocking）。
4. **视觉连线测试**：利用 Deck.gl 的 `TripsLayer` 图层跑这些模拟数据。
**▶ 验收里程牌**：运行本地服务后浏览器不报错，屏幕上出现了中国地形成熟切片，且历史点位之间成功亮起发光轨迹。

---

### 🏭 Phase 1: “无限细化”工业级数据机床
*当前状态：等 Phase 0 的 MVP 页面跑通后接棒。*

**核心目标**：从人工对话彻底转向自动化车间，让 AI 管线具有死板甚至冰冷的军事级执行纪律。
**具体拆解动作**：
1. **`skills/` 基建**：确立 10 余个方志专属大模型提示词文件体系。里面强制囊括《AI_RULES.md》中必须使用的 〖=地名〗、〖$物产〗 等多维专属符号要求。
2. **写中枢大脑**：建立 `pipeline_runner.py` 自动化读取文件并呼叫大模型工作的批处理总控室脚本。
3. **质检门禁拦截**：写入 `lint_integrity.py` 保留原文字节门限程序。如果经过 AI 提取之后原始史书字节受损一丁点，坚决回滚，永远堵塞在 `TotalData` 入库大门外。
**▶ 验收里程牌**：丢进一段《明史》文言文到底料文件夹，执行 Python，控制台自动跑出完美断句加标记的源码体。

---

### 🧭 Phase 2: 空间时空挂载与消歧中枢
*当前状态：结合原始 JSON 与外部 GIS 地理库对接定点。*

**核心目标**：解决“历史地名人名没有精确物理坐标系”的最致命缺陷，完成千年古城表象与地理经纬实质的无缝缝合。
**具体拆解动作**：
1. **词典入库与提纯**：手动将极其笨重的 CHGIS（或其他大型古地名词典包）丢入 `TotalData/GISData/`，使用 AI 撰写的超暴力 ETL 清洗脚本将几百兆脱水成几兆极简版坐标字典 `gazetteer_dict.json`。
2. **缝合挂载器开发**：开发核心脚本引擎 `entity_linker.py`。
3. **全网时空打通**：引擎将 Phase 1 里抽出来的“地名+朝代”强行去挂载字典库里碰撞匹配，赋予实地坐标系，生成最终所需的 `persons.json`、`events.json`。
**▶ 验收里程牌**：虚无的“建康”、“江宁”，在 JSON 文件里变身成了携带具体生成时间戳和精准 ID、经纬度的物理锚点。

---

### 🎥 Phase 3: 全局拼装与影视级成品展示
*当前状态：展示面板的终战归拢。*

**核心目标**：经脉全面贯通，展示出终极数字人文赛博书库。
**具体拆解动作**：
1. **阅读器重构点亮**：解析 JSON 切片，并在 Vue 3 左侧将古籍原文彻底重拼出来，生成高亮排版页面。
2. **Event Bus 事件打通**：开发双端通讯。鼠标在左侧点击实体名，右侧 3D 镜头平滑加速 `flyTo` 突降特写目标点。
3. **年轮底盘拖拽滤镜**：加装全局时间轴，基于 JSON 中的年份与朝代属性，随滑块拖拉实现发光点的沧海桑田更替展现。
4. **最后封板屏蔽**：确保 `TotalData` 永远被 `.gitignore` 拦截不传外网，编写使用指南。
**▶ 验收里程牌**：整体交互大满贯，项目可独立部署、独立宣发开源。
