# 历史知识信息系统总架构文档 (System Architecture Master Plan)

## 1. 系统愿景与定位

“无限时空图”项目已从单一的 3D 地图组件，全面升维为**立体历史知识体系的前端呈现底座**。
本系统旨在通过大模型提取的结构化历史数据，为历史地理学、博物馆学、古典文献学提供多维度、跨模块联动的探索环境。

## 2. 核心架构模式：多维路由与单一事实来源

系统采用 **SPA (单页应用) + Vue Router** 架构，彻底告别原本拥挤的单页面设计。
所有数据依赖于后端的离线九步管线预生成的 `ProcessedData/` 目录下的 JSON 文件，前端遵循“纯展示与联动逻辑”的轻量化原则。

## 3. 四大核心子系统 (The Four Pillars)

整个前端系统由四大平级的业务系统构成，通过左侧全局 Sidebar 进行统一导航调配。

### 3.1 📖 典籍精读系统 (Academic Reader)
**职责**：提供源头文献的深度阅读与考据环境。
- **多列比对引擎**：支持 `[影印件] | [繁体原貌] | [简体译文]` 三列/双列并排同步滚动对比，满足严谨考据需求。
- **实体高亮穿透**：所有被大模型标注的人、地、物、官职等实体全量高亮，点击可呼出实体微型信息卡片。
- **格式包容性**：兼容古代方志、现代 PDF 论文、Markdown 资料等多种文本源。

### 3.2 🏛️ 泛实体百科系统 (Universal Wiki)
**职责**：全面详尽地展示单一实体的全生命周期数据。
- **万物皆可百科**：涵盖人物（生平/著作）、地理建置（沿革史）、文博遗址、历史事件、官制法度等。
- **标准模块化版式**：采用 Wikipedia 风格，左侧为 Infobox（核心属性表），右侧为正文与事迹梳理。
- **关系网跃迁锚点**：词条页内嵌显眼的【溯源关系网】按钮，点击无缝切入图谱系统。

### 3.3 🧠 知识图谱系统 (Knowledge Graph)
**职责**：梳理与展示历史实体间错综复杂的拓扑关系。
- **底层引擎**：基于 ECharts / D3.js 等关系图渲染库。
- **多维关系映射**：展现家族血缘树、政治党争派系、师生学术传承、甚至藏品流转网络。
- **双向联动**：在图谱中选中某个节点，侧边栏或弹窗立即呈现对应的“实体百科”简报，实现宏观与微观的来回穿梭。

### 3.4 🌐 宏观时空沙盘 (SpatioTemporal Map)
**职责**：专门处理具备绝对时空特征（经纬度 + 时间戳）的数据渲染。
- **底层引擎**：MapLibre GL JS + Deck.gl。
- **微观/宏观双模态时间轴**：支持全局两千年维度的宏大叙事，亦支持点击单一人物后自动放大至月份粒度的生平流转。
- **多图层渲染**：保留原有的家族迁徙弧线、人物轨迹、战争热力等核心时空表达方式。

## 4. 全局状态与系统联动 (State & Linkage)

模块间并非孤立存在，而是由一套全局的事件总线（Event Bus 或 Pinia 状态树）串联。

**核心联动场景 (Cross-module Linkage)**：
1. **从典籍到图谱**：在《王阳明传》阅读模式中点击“王守仁”，系统状态 `activeEntity` 更新。如果此时用户切换到【知识图谱】路由，图谱会自动以王阳明为中心节点展开政治关系网。
2. **从百科到地图**：在【实体百科】查看“修文县（龙场驿）”时，点击【查看时空位置】，系统自动跳转至【时空沙盘】路由，镜头平滑 `flyTo` 贵州，并高亮其沿革轨迹。

## 5. 前端目录结构规范

```text
app/
├── public/                 # 静态资源
├── src/
│   ├── assets/             # 样式与图片
│   ├── components/         # 跨页面复用组件 (如 TimelineSlider.vue)
│   ├── layouts/            # 页面布局骨架
│   │   └── SystemLayout.vue # 包含全局导航侧边栏的外壳
│   ├── router/             # 路由配置
│   │   └── index.js
│   ├── store/              # 全局状态管理 (Pinia)
│   ├── views/              # 四大核心路由页面
│   │   ├── MapView.vue     # 🌐 时空沙盘
│   │   ├── ReaderView.vue  # 📖 典籍精读
│   │   ├── WikiView.vue    # 🏛️ 实体百科
│   │   └── GraphView.vue   # 🧠 知识图谱
│   ├── data/               # 存放解析好的静态 JSON
│   ├── App.vue             # 根级入口 (仅包含 router-view)
│   └── main.js             # Vue 实例挂载
```
