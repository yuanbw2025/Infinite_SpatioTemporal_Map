# 无限时空图 - AI Agent 绝对工作宪法 (Agent Core Directives)

> **警告**：任何接手此项目大模型管线提取任务的 AI Agent（包括 Claude Code, Cursor, Aider 等），在执行相关任务前，必须无条件遵循以下最高指令。

## 📍 行动准则第一条：绝对的数据原真性 (Zero-Tampering)
无论要求你做何种翻译、校对、结构化分析，**严禁修改、删除、替换底本原文中的任何一个汉字或标点符号**。
你的操作权限仅限于：在原文中插入规范的「本体提取符号序列」。
- ❌ 错误做法：原文是“太守张公”，你改成了“太守张公（张三）”。
- ✅ 正确做法：原文是“太守张公”，你改为了“〖§太守〗〖@名宦:张公〗”。

## 📍 行动准则第二条：服从方志专属本体规范
禁用你预训练库中的“人物/地点/机构”模糊标签。必须使用硬性规定的专属提取符：
- 人名类：`〖@分类:名称〗` (如 `〖@乡贤:王二〗`, `〖@列女:李氏〗`)
- 地理区划类：`〖=区划名称〗` (如 `〖=江夏郡〗`)
- 城池防区：`〖♜防区〗` (如 `〖♜雁门关〗`)
- 聚落微地名：`〖⌂聚落〗` (如 `〖⌂乌衣巷〗`)
- 机构公署：`〖#衙署〗` (如 `〖#兰考县衙〗`)
- 灾害异象：`〖!灾异〗` (如 `〖!大雪〗`, `〖!蝗灾〗`)
- 经济物产：`〖$特产〗` (如 `〖$蜀布〗`)
- 课税徭役：`〖¥课税〗` (如 `〖¥秋粮〗`)
- 时间纪年：`〖⌚时段〗` (如 `〖⌚万历十五年〗`)
未在名单上的特殊标签，宁可不打，不得自创。

## 📍 行动准则第三条：严禁地理/时间幻觉脑补
在由提取阶段转向数据组装阶段（生成 JSON）时：
如果你不知道某个古地名的绝对经纬度，或者不知道某个朝代年号对应的公元纪年，**请保留 Null 或原名，强烈禁止使用大模型去进行幻觉猜测。** 所有经纬度和时间换算交由后期的 Python 脚本去读取专业的外部 CHGIS 等知识库来自动完成。你只做“剥离者”，不做“发明家”。

## 📍 行动准则第四条：单一技能聚焦 (Single-Skill Precision)
我们采用“柳叶刀方法”进行无限细化。不要试图“一步到位”。
如果要求你执行 `SKILL_03_实体提取`，你就只负责提取实体，**绝对不要**顺手去做翻译或者关系连线。一次提示词回合，只干一件微观的事。
