# 方志本体实体提取技能 (Gazetteer Entity Extractor)

## 技能描述
基于《无限时空图》本体规范，从地方志文言文底本中精准提取人物、地理、机构、物产、兵事等核心实体，并使用专属符号进行标注。

## 核心任务
1. **实体识别**：识别文本中的所有实体。
2. **符号嵌套**：将识别出的实体包裹在对应的专属符号中。
3. **原文一致性**：严禁修改原文任何字符（包括标点）。

## 标注规范 (引用自 AI_RULES.md)
- **人物**：`〖@群体:名字〗`（如 `〖@乡贤:王守仁〗`）
- **区划**：`〖=地名〗`（如 `〖=江夏郡〗`）
- **自然地理**：`〖~水火〗`（如 `〖~云梦泽〗`）
- **防务城池**：`〖♜关津〗`（如 `〖♜雁门关〗`）
- **聚落**：`〖⌂村名〗`（如 `〖⌂乌衣巷〗`）
- **物产**：`〖$特产〗`（如 `〖$蜀布〗`）
- **纪事**：`〖⚔兵事〗`, `〖!异象〗`

## 执行指令
请阅读输入文本，直接输出带有上述符号标注的 Markdown 文本。不要进行任何摘要，不要省略任何原文段落。

### 示例输入
嘉靖二年，知府苏轼重修岳麓书院。其地产蜀布，闻名天下。

### 示例输出
〖⌚嘉靖二年〗，〖§知府〗〖@名宦:苏轼〗重修〖^教育:岳麓书院〗。其地产生〖$物产:蜀布〗，闻名天下。
